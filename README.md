# Cellcard Telecom Landing

A responsive landing page built with HTML5, Tailwind CSS, and vanilla JavaScript.

## Features
- Sticky navigation bar with mobile hamburger menu
- Hero section with background images (Swiper) + headline and two CTAs
- "Why Cellcard" section with 6 feature cards using CSS Grid
- Pricing section with exactly 3 plans (Free, Premium Individual, Family) using Grid/Flex
- Testimonials section (Swiper carousel) with at least 3 items
- Footer with navigation links, social icons, and a small signup form
- Interactive elements: mobile menu toggle, language bar hides on scroll, CTA hover/active states, smooth scroll, carousel controls

## Assets
- Place all images in `assets/images/`
- Include `favicon.ico` in the project root

## Run (Methods)
Choose one of the methods below to run the project locally.

1) Method 1 — Open file directly

	- Double-click `index.html` or open it in your browser.

2) Method 2 — Simple Python HTTP server (PowerShell)

	- Run in the project folder:

	```powershell
	python -m http.server 8080
	```

	- Then visit: http://localhost:8080/

3) Method 3 — Simple Node static server

	- If you have `npx`/Node installed, run from the project folder:

	```powershell
	npx serve .
	```

	- Then visit the printed URL (usually http://localhost:5000 or similar).

4) Method 4 — VS Code Live Server (recommended for development)

	- Install the `Live Server` extension from the VS Code marketplace.
	- Open the workspace in VS Code, right-click `index.html` and choose "Open with Live Server", or click the "Go Live" button in the status bar.
	- Live Server opens the site in your browser (commonly at http://127.0.0.1:5500/).
