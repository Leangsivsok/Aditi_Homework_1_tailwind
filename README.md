# VibeFlow Telecom Landing

A responsive landing page built with HTML5, Tailwind CSS (via CDN), and vanilla JavaScript.

## Features
- Sticky navigation bar with mobile hamburger menu
- Hero section with background images (Swiper) + headline and two CTAs
- "Why VibeFlow" section with 6 feature cards using CSS Grid
- Pricing section with exactly 3 plans (Free, Premium Individual, Family) using Grid/Flex
- Testimonials section (Swiper carousel) with at least 3 items
- Footer with navigation links, social icons, and a small signup form
- Interactive elements: mobile menu toggle, navbar color change on scroll, CTA hover/active states, smooth scroll, carousel controls

## Assets
- Place all images in `assets/images/`
- Ensure images are under 200 KB (use tools like Squoosh or TinyPNG to compress)
- Include `favicon.ico` in the project root (a placeholder is included; replace with your own 16x16 or 32x32 icon)

## Run
Open `index.html` directly in your browser. If you use a local server:

```
# Windows PowerShell (simple server via Node or Python)
# Python 3
python -m http.server 8080
# Or Node (if installed globally)
npx serve .
```

Then visit `http://localhost:8080/`.
